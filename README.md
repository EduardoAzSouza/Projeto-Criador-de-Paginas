Projeto Criador de Páginas — Exemplo de uso local
===============================================

Resumo
- Exemplo simples: este projeto gera páginas (landing pages) através de chamadas a uma API de IA.

Como usar (exemplo rápido)
1. Abra `Index.html` no navegador.
2. Insira sua chave API localmente seguindo uma das opções abaixo (NUNCA comite a chave em um repositório público).

1. Adicione a API key ao `scripts.js`

- Abra `scripts.js` e substitua o placeholder `<TOKEN AQUI>` pela sua chave **somente localmente**.

Placeholder abaixo (substitua pelo seu token localmente):

```
<TOKEN AQUI>
```

AVISO IMPORTANTE
- Eu não posso inserir seu token real neste repositório por motivos de segurança. Colocar a chave em arquivos públicos (incluindo `README.md`) a torna acessível a qualquer pessoa.

Resumo rápido dos arquivos
- `Index.html` — interface
- `scripts.js` — lógica que chama a API (contém o placeholder `<TOKEN AQUI>`)
